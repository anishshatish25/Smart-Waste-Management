# Urban Grid AI Logistics Engine

An intelligent IoT-based urban logistics optimization engine that monitors 1,000+ city bins, identifies overflow hazards, and orchestrates fleet dispatch routes in real-time.

## 🛠 Tech Stack
- **Backend:** Python, FastAPI, NumPy, Pandas, Matplotlib, Seaborn
- **Frontend:** HTML5, Tailwind CSS, Leaflet.js
- **Intelligence:** Spatial Quadrant Math, Predictive Urgency Indexing
